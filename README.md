## INSTALLATION

```

```


## UTILISATION

Pour scanner un site en .onion copier/coller dans un terminal

```
sudo service tor start && sudo service privoxy start && onionscan --torProxyAddress=127.0.0.1:9050 http://lesite.onion
```
